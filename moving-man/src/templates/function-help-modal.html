About Custom Functions

You can specify an expression where position is a function of time (t).


Examples of valid expressions:

  • ƒ(t) = 3 * t + 2
  • ƒ(t) = 2*sin(t)
  • ƒ(t) = pow(t,2)
  • ƒ(t) = 7*pow(sin(t),2)
  • ƒ(t) = sqrt(t * 4)

Note that you only input the right side of the equation and also that multiplication must be specified explicitly as in 3 * t, instead of implicitly as in 3t.


Available math functions:

Function Description
abs(x) absolute value of x
acos(x) arccosine of x
asin(x) arcsine of x
atan(x) arctangent of x
cos(x) cosine of x
exp(x) Ex, where E is Euler's constant
log(x) natural logarithm of x
pow(x, y) xy
sin(x) sine of x
sqrt(x) square root of x
tan(x) tangent of x

Available math constants:

Constant Description
PI Math constant π and ratio of circumference of a circle to its diameter
E Euler's constant and the base of natural logarithms
